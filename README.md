# Task 1 – Data Cleaning and Preprocessing

## Dataset
Netflix Movies and TV Shows Dataset

## Objective
Clean and preprocess the raw dataset, which contains missing values, duplicates, and inconsistent formats.

## Steps Performed
- Identified missing values using isnull()
- Replaced high-null categorical columns with "Unknown."
- Dropped rows with minimal missing values
- Converted date_added to datetime format
- Removed duplicate rows
- Standardized text formatting
- Cleaned column names
- Verified data types and final dataset quality

## Final Dataset Shape
8702 rows × 12 columns

## Tools Used
Python (Pandas), Jupyter Notebook, and Excel

The dataset is now cleaned and ready for analysis or modeling.
